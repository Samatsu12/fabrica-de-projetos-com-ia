# Modelos de CLAUDE.md

O `CLAUDE.md` é o **manual interno do projeto** — o Claude Code lê esse arquivo
automaticamente toda vez que você trabalha dentro daquela pasta. É onde ficam as instruções
fixas, para você não repetir tudo a cada conversa.

## Como usar
1. Escolha o modelo do tipo de projeto mais parecido com o seu.
2. Copie o conteúdo para um arquivo chamado `CLAUDE.md` na **raiz** do seu projeto.
3. Preencha os campos `[entre colchetes]`.
4. (Atalho) Ou rode `/gerar-claudemd` e deixe a skill montar para você.

| Arquivo | Para que tipo de projeto |

| [`generico.CLAUDE.md`](generico.CLAUDE.md) | Qualquer projeto (ponto de partida) |
| [`landing-page.CLAUDE.md`](landing-page.CLAUDE.md) | Páginas de venda / captação |
| [`crm.CLAUDE.md`](crm.CLAUDE.md) | CRM e controle de leads |
| [`dashboard.CLAUDE.md`](dashboard.CLAUDE.md) | Painéis de indicadores |
| [`automacao-planilha.CLAUDE.md`](automacao-planilha.CLAUDE.md) | Relatórios a partir de CSV/Excel |

> Cada projeto na pasta `projetos/` já vem com um `CLAUDE.md` preenchido de exemplo.
