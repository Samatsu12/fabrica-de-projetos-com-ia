# Projeto 01 — Landing Page para Prestador de Serviço

## Para quem serve
Profissionais que vendem serviço e precisam de mais contatos pelo WhatsApp: eletricista,
encanador, clínica de estética, dentista, personal, consultor, advogado, salão, assistência
técnica, empresa de limpeza, manutenção residencial.

## A dor
Depende de indicação e de um Instagram desorganizado. Quando o cliente pergunta "você tem um
site?", não há uma página clara que apresente o serviço e gere confiança. Isso faz perder venda.

## Primeira versão útil (o que ENTRA)
- Dobra inicial com promessa clara + botão de WhatsApp
- Explicação e lista de serviços
- Benefícios de contratar
- Bloco de confiança (depoimentos, números, fotos)
- Chamada para ação
- Perguntas frequentes
- Botão fixo de WhatsApp

## Fora do escopo agora
Login, pagamento online, área do cliente, blog, agendamento avançado.

## Passo a passo (no Claude Code)
1. Copie o [`CLAUDE.md`](CLAUDE.md) deste projeto para a pasta do seu projeto e preencha.
2. Rode `/landing` (ou cole o prompt principal abaixo).
3. Aprove a estrutura que a IA apresentar.
4. Peça os textos com o subagent **copywriter** e o capricho visual com **ux-ui-designer**.
5. Teste com `/qa` (inclusive no celular).
6. Revise com `/seguranca` e publique com `/deploy-check`.

## Prompt principal
```
Quero criar uma landing page para um prestador de serviço local.

Tipo de serviço: [ex.: eletricista, clínica de estética, assistência técnica]
Público-alvo: [descreva quem contrata]
Objetivo da página: gerar confiança e levar o visitante a chamar no WhatsApp para pedir
orçamento ou atendimento.

A página precisa ter: dobra inicial com promessa clara; breve explicação do serviço; lista
dos principais serviços; benefícios de contratar; bloco de confiança; chamada para ação;
botão de WhatsApp; perguntas frequentes; layout simples, moderno e responsivo.

Tom: profissional, direto e confiável. Sem exageros e sem prometer resultado impossível.
Evitar: login, painel, pagamento ou blog nesta primeira versão.
Critério de sucesso: a pessoa entende o serviço rapidamente e se sente segura para chamar.

Antes de criar os arquivos, me mostre a estrutura da página e explique a função de cada bloco.
```

## Prompt de melhoria
```
Revise esta landing page com foco em clareza e conversão. Avalie: a headline está clara? O
visitante entende rápido o serviço? O botão de WhatsApp aparece nos momentos certos? Tem
texto genérico? Falta bloco de confiança? O visual está simples para um público comum? O que
remover? O que melhorar? Depois, proponha uma versão melhorada da estrutura e dos textos.
```

## Como vender este projeto
"Percebi que muitos clientes chegam pelo WhatsApp, mas antes de chamar precisam confiar. Uma
página simples apresenta o que você faz, mostra seus diferenciais e facilita o orçamento."

**Preço de referência:** simples R$ 297–497 · com copy trabalhada R$ 597–997 · com criativos
e configuração de anúncio R$ 1.200+.
