# 12 Projetos Práticos para Pequenos Negócios

Cada pasta é um projeto pronto para você **gerar com o Claude Code**. Não vem com o código
pronto de propósito — o valor é você aprender a construir conduzindo a IA (esse é o método).

Cada projeto tem:
- **README.md** — para quem serve, a dor, a primeira versão útil, o passo a passo e o prompt principal.
- **CLAUDE.md** — o manual interno pronto para copiar para a pasta do seu projeto.

## Como usar um projeto
1. Crie uma pasta nova para o seu projeto (ex.: `landing-do-joao`).
2. Copie o `CLAUDE.md` do projeto escolhido para dentro dela e preencha os `[colchetes]`.
3. Rode `claude` dentro da pasta e cole o **prompt principal** do README.
4. Siga o fluxo: aprovar estrutura → gerar → testar (`/qa`) → segurança (`/seguranca`) → publicar (`/deploy-check`).
