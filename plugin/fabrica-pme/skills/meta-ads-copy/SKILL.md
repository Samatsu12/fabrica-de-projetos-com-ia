# Meta Ads Copy — criativos que vendem

Gera **conjuntos de anúncios prontos para testar**, no formato que as plataformas pedem,
com variações para o usuário testar e descobrir o que funciona.

## Quando usar
- O projeto (landing/oferta) está no ar e o usuário quer atrair clientes pagos.

## Procedimento

### 1. Reúna o essencial
- Produto/serviço e oferta exata (o que a pessoa ganha?).
- Público-alvo e a dor principal.
- Diferenciais e provas.
- Ação desejada (chamar no WhatsApp, comprar, preencher form).
- Tom da marca.

### 2. Gere os criativos no formato das plataformas
Para **Meta Ads (Facebook/Instagram)**, entregue por ângulo de abordagem:
- **Texto primário** (3 variações: dor, desejo, prova/urgência) — 1 a 3 frases curtas.
- **Headline** (5 variações curtas, até ~40 caracteres).
- **Descrição** (2 variações).
- **CTA** sugerido (ex.: "Enviar mensagem", "Saiba mais").

Para **Google Ads** (se pedido): 5 headlines (até 30 caracteres) + 2 descrições (até 90).

### 3. Sugira ângulos de teste
Monte pelo menos 3 ângulos diferentes (ex.: economia de tempo, medo de errar, prova
social) — porque o segredo de tráfego é **testar variações**, não acertar de primeira.

### 4. Roteiro de criativo (vídeo/imagem)
Para cada ângulo, descreva uma ideia de criativo: gancho nos 3 primeiros segundos,
desenvolvimento, CTA. Inclua sugestão de imagem/cena.

### 5. Organize para uso
Entregue numa tabela ou lista clara que o usuário possa copiar direto para o Gerenciador
de Anúncios.

## Cuidados
- **Respeite as políticas das plataformas:** nada de promessa de cura, renda garantida,
  "antes e depois" proibido (saúde/estética), ou afirmação sem prova.
- Linguagem honesta. Exagero queima verba e reprova anúncio.
- Adapte o tom ao público; não use o mesmo texto para todo nicho.
